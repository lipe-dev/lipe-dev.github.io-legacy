### How to use this project:
- Clone it with 
```sh
git clone git@github.com:lipe-dev/lipe-dev.github.io.git
```
- Set everything up with 
```sh
yarn install
```
or 
```sh
npm install
``` 
- Run it with 
```sh
yarn dev
```
or 
```sh
npm run dev
```
